About the Project


Sentiment Wars is a sentiment analyzer of tweets regarding 'Star Wars: The Force Awakens'. In this web app, you will see a series of tweets queried using the Twitter Search API and the Tweepy wrapper, along with their sentiment analysis using a 4-state finite automaton. The actual automaton is generalized, i.e. it can accept any input in the form of multiple English words delimited by whitespace. The Sentiment Wars automaton can be downloaded from this app's Github, along with the README containing directions on how to run the automaton from the command line with user-inputted tweets, or with specific Twitter Search queries.
